ChefBoyRD Documentation
=======================

**ChefBoyRD is a restaurant development platform designed to launch food businesses into the 21st century.**

.. image:: https://travis-ci.org/ZacBlanco/ChefBoyRD.svg?branch=master

.. image:: https://img.shields.io/badge/Python-3.3%2B-orange.svg

.. image:: https://img.shields.io/badge/license-AGPL-blue.svg

See below for more information on our project

 - `GitHub Repository <https://github.com/zacblanco/chefboyrd>`_

This project was created as a part of a software engineering class (14:332:452) in the Spring of 2017 `taught by Professor Ivan Marsic <http://www.ece.rutgers.edu/~marsic/Teaching/SE/>`_

Please see the `project readme <https://github.com/zacblanco/chefboyrd>`_ for instructions on how to setup and install the application.
